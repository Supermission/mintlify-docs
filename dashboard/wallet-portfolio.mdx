# Wallet portfolio

The primary dashboard view shows your complete portfolio across all connected chains with real-time balances, mini price charts, and multi-wallet support.

## Multi-wallet support

You can monitor up to 3 wallets simultaneously:

| Feature | Description |
|---------|-------------|
| **Active wallet** | Your primary connected wallet. |
| **Watch-only wallets** | Add any wallet address for monitoring (no trading). Click "Watch" → enter address → add. |
| **Switching** | Click any wallet pill to make it the active view. All data below updates. |
| **Removing** | X button on non-primary wallets to remove them. |
| **Copy** | Copy button for the active wallet address. |
| **Explorer** | Direct link to view your wallet on the block explorer. |

## Balance section

| Element | Description |
|---------|-------------|
| **Total Portfolio USD** | Aggregate portfolio value across all chains |
| **24h Change** | Percentage change with green/red color and trend arrow |
| **Holdings count** | Number of distinct tokens held |
| **Chains count** | Number of chains with assets |

## Chain balances

A grid of chain cards, each showing:

| Chain | Data shown |
|-------|-----------|
| **Base** | ETH native balance + total USD value for all Base tokens |
| **Polygon** | Native balance + total USD value |
| **Ethereum** | ETH native balance + total USD value |
| **Optimism** | ETH native balance + total USD value |

## Token holdings list

Every token in your portfolio, displayed as clickable rows:

| Element | Description |
|---------|-------------|
| **Token icon** | With chain badge showing which chain the token is on |
| **Token symbol** | e.g., USDC, WETH, AERO |
| **Chain name** | Which chain this token is on |
| **Mini price chart** | Small chart showing recent price history (green if up, red if down). Hidden for stablecoins. |
| **USD value** | Token balance in USD |
| **Token balance** | Raw token amount |
| **Portfolio %** | Percentage of total portfolio |

Click any token row to expand and see a full interactive price chart with period selector (24h, 7d, 30d, 90d) and color-coded direction.

## AI wallet analysis

Below the portfolio overview, you can get on-demand AI analysis of your wallet:

- Click **"Analyze Wallet"**
- Get back:
  - **Portfolio Health Score** — Excellent/Good/Fair/Poor
  - **Risk Score** — 0-100, color-coded
  - **Diversification Rating**
  - **Concentration Risk** — Low/Medium/High
  - **Top Holdings summary**
  - **Recommendations** with actionable advice

<Info>
  Wallet data refreshes automatically every 15 seconds when the page is visible.
</Info>
