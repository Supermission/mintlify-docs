# Meet your DeFi agents

DeFAI is your chat-based DeFi terminal. Five specialized AI agents, each built for a different job. You talk to them in plain English. They do the heavy lifting.

Want to know if a token is safe? Ask. Need to find the best yield on your USDC? Ask. Want to swap tokens across chains? Just say the word.

## How it works

1. **Open a chat** — pick the agent you need (or just start typing and we'll route you)
2. **Ask your question** — use natural language, like talking to a smart friend who knows DeFi inside out
3. **Get results** — the agent researches, analyzes, and gives you a clear answer with actionable next steps
4. **Execute in one click** — swaps, deposits, and bridges happen right inside the chat. No switching apps.

## What happens when an agent acts

When an agent needs to do something on-chain — like execute a swap, deposit into a yield vault, or bridge tokens — you'll see a progress card right in the chat. It walks you through each step:

1. **Checking** — the agent verifies the transaction details
2. **Approving** — you approve the token spend (if needed)
3. **Executing** — the transaction is submitted
4. **Done** — you get a transaction link to verify on the blockchain

You're always in control. Nothing happens without your approval.
