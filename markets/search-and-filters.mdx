# Search and filters

With thousands of active markets, you need good tools to find what matters. The sidebar on the left gives you everything you need to narrow things down.

## Categories

Click any category to filter the grid:

| Category | What's in it |
|----------|--------------|
| **All Markets** | No filter — everything |
| **Politics** | Elections, legislation, government actions |
| **World** | Geopolitics and international events |
| **Crypto** | Token prices, protocol events, chain milestones |
| **Sports** | Game outcomes, championships, player stats |
| **Finance** | Interest rates, economic indicators, corporate events |
| **Tech** | Product launches, company milestones |
| **Culture** | Entertainment, media, social trends |
| **Science** | Research, space, climate, health |
| **More** | Everything else |

Click a category again to deselect it and go back to All Markets.

## Sort options

Control the order of market cards with the sort dropdown:

| Sort | When to use it |
|------|----------------|
| **Volume** (default) | See the most-traded, most-liquid markets first. Best prices and easiest fills. |
| **Trending** | Markets with sudden spikes in activity. Something just happened or is about to. |
| **Newest** | Recently created markets. Fresh opportunities that may not be fully priced yet. |
| **Ending Soon** | Markets closest to resolution. Time-sensitive — these are about to pay out. |
| **Movers** | Biggest price changes in the last 24 hours. Where the action is right now. |

## Search

Type any keyword to search market questions. Results update as you type. Clear the search box to go back to the full filtered view.

## View mode

Switch between two layouts:
- **Normal** — every market gets its own card in a flat grid
- **Stacked** — markets grouped by parent event (see [Stacked View](/markets/stacked-view))

## On mobile

On smaller screens, the sidebar is tucked away. Search and sort controls appear at the top of the market grid. Categories are available through a collapsible menu.

## Combining filters

All filters work together. Some powerful combinations:

- **Crypto** + **Trending** = crypto markets with unusual activity right now
- Search "Fed" + **Ending Soon** = Federal Reserve markets about to resolve
- **All Markets** + **Movers** = the biggest price swings across every category
- **Sports** + **Newest** = freshly listed sports markets

Changing any filter takes you back to the top of the results.

<Info>
  The Signals tab has its own separate filters (signal type, category, confidence threshold, etc.). Those don't affect the Browse tab and vice versa.
</Info>

<Tip>
  Start with **Trending** to see what's moving, then switch to **Volume** to find the deepest liquidity for execution. Use search when you have a specific topic in mind.
</Tip>
